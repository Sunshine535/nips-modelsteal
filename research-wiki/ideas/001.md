# Transformer Tomography: Observability-Governed Progressive Parameter Inversion

## Hypothesis

The Observability Gramian G(Q) = (1/N)Σ J_i^T J_i, computed on the symmetry-quotiented parameter space of a transformer suffix, governs the recoverability of internal weights from black-box logit access. A single unified framework — not three stapled papers — where observability theory predicts, guides initialization, and optimizes queries for parameter recovery.

## Core Components

1. **Observability Gramian on Quotient Space**: Define G(Q) using Jacobian-vector products (JVPs) projected onto the orthogonal complement of continuous symmetry tangent directions (RMSNorm scaling, FFN neuron scaling). The eigenspectrum of G predicts per-layer recovery difficulty.

2. **Sketched Gauss-Newton Algebraic Initialization**: Use JVP probes to build a reduced Gramian, solve a truncated ridge regression to initialize along observable directions. Replaces random initialization.

3. **Fisher-Optimal Query Design**: Optimize synthetic prompts to maximize λ_min(projected G), ensuring the query set illuminates the hardest-to-observe directions.
